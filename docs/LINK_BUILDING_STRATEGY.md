### Postup

1. **Ahrefs Webmaster Tools (free):** backlink profil konkurentů
2. **Check My Links (Chrome):** projít top stránky linkující na konkurenty
3. Najít 404/broken → nabídnout náš obsah jako náhradu
4. **Wikipedia:** vytvořit článek "Výkup nemovitostí" (nofollow, ale autorita)
