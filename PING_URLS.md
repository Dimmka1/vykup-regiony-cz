# Search Engine Ping URLs

After deploying new blog content, ping these URLs to notify search engines:

## Google

```
https://www.google.com/ping?sitemap=https://vykoupim-nemovitost.cz/sitemap.xml
```

## Bing (IndexNow)

```
https://www.bing.com/indexnow?url=https://vykoupim-nemovitost.cz/blog/feed.xml&key=<YOUR_INDEXNOW_KEY>
```

If IndexNow key is not configured, submit sitemap manually via Bing Webmaster Tools.
